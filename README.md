# kanban-board

Реализация канбан доски на чистом JS

###### Возможности:
* Добавлять карточки
* Добавлять колонки
* Переносить карточки между колонками

###### Демонстрация
Посмотреть на приложение можно, перейдя по [ссылке](https://astr0x.github.io/kanban-board).
